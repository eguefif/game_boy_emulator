# Gameboy emulator in rust

## Description

This gameboy emulator is a practice project to learn rust and how the hardware works.

## Getting Started

### Dependencies

- Rust

### Installing

- git clone the repo

### Executing program

```bash
cargo run -- YOUR ROM
```
